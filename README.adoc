= sonarlint4netbeans

== SonarLint integration for Apache Netbeans

Features:

- Support Java and Javascript
- Annotation in editor
- Type SonarLint in Action Items
- Enable/Disable Rules in Tools/Options/Miscellaneous/SonarLint
